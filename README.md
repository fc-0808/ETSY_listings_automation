# Listings automation (private)

Private seller tooling for **my own Etsy shops**: structured listing manifests, optional AI-assisted draft copy, and integration with the **Etsy Open API** to create or update drafts—reviewed before publish.

This repository exists as the project home for development and for app registration context. It is **not** a public product and is **not** offered to third parties.

**No API keys, tokens, or secrets belong in this repo** (use environment variables or your OS credential store).
